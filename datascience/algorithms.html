Professional Experience

More recently, I've needed to solve a variation of the knapsack problem with my work for Hearst Digital's newspaper platform. While not explicitly presented as an NP-complete problem, the problem called for a solution to eloquently choose a template based on the number of stories in a newspaper section. I was able to recognize the complexity of the problem at hand, given the large number of templates, and provide a simple, recursive solution, as can be seen below:



- (void)iterateOver:(NSArray *)templateTypes toCreate:(NSMutableArray *)templateArray withTotal:(int)total andStoryCount:(int)storyCount {
    for(NSNumber *n in templateTypes) {
        total += [n intValue];
        if(total == storyCount) {
            [templateArray addObject:n];
            NSSortDescriptor *lowestToHighest = [NSSortDescriptor sortDescriptorWithKey:@"self" ascending:YES];
            [templateArray sortUsingDescriptors:[NSArray arrayWithObject:lowestToHighest]];
            [_templateOptions addObject:templateArray];
        }
        if(total < storyCount) {
            [templateArray addObject:n];
            NSMutableArray *templateArrayCopy = [[[NSMutableArray alloc] initWithArray:templateArray copyItems:YES] autorelease];
            [self iterateOver:templateTypes toCreate:templateArrayCopy withTotal:total andStoryCount:storyCount];
        }
    }
}
