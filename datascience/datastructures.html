Data Structures

Academic Experience

I was introduced to the concepts of Data Structures at Christopher Newport University as a computer science student in CPSC 320. While I was already aware of primitives and simpler abstract data types, it was in this class that I learned of more complex data structures, such as linked lists, binary trees and graphs, as well as the benefits of using data structures for efficiency and reduction of maintenance. Also, since most of my programming assignments were completed in C++ at the time, I became aware of the importance of memory management when dealing with collections and associations. Looking back, this course laid the foundations for much of what I do today.


I received an A (12 credit hours) for CPSC 320.


Professional Experience

In my career as a web and mobile software engineer, I use data structures nearly every day. For instance, when developing in Objective-C for iOS, I commonly use the complex data structures provided by Cocoa; and in my Java based applications, I routinely make use of parameterized typed data structures, which uses generics to type-check at compile time.


I also understand the importance of creating object classes. Extending basic data structures into objects has been invaluable when working on large-scale applications, and utilizing object-oriented techniques, such as abstraction, has allowed me to manage all levels of interactions in a program, regardless the complexity.
